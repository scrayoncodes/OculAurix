OccluAuric is a lightweight accessibility tool designed to support people with hearing or speech impairments.  
It offers **Speech-to-Text**, **Text-to-Speech**, a **Kill Switch**, and **Communication History**, all running through a simple Flask backend and browser-based interface.

---

## Features

### Text-to-Speech (TTS)
Converts typed text into spoken audio using an offline engine.

### Speech-to-Text (STT)
Transcribes speech into text using a microphone.

### Kill Switch
Instantly stops or blocks all STT/TTS activity so the user has full control.

### Communication History
Saves every interaction (STT, TTS, ASL if enabled) to `history.json` for later reference.
